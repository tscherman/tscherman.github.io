Work

In this project I explored the academic outcomes of over 2,000 students using a dataset from Kaggle. The primary goal is to identify key factors that influence final grades and offer insights to help students improve their academic performance. The project involves data cleaning, in-depth exploratory data analysis (EDA) with interactive visualizations via Plotly, and the development of predictive models using K-Nearest Neighbors (KNN), Random Forest, and Logistic Regression. The results are presented through an interactive Streamlit web application for easy exploration and understanding
